Simplified for Loop Syntax in VisionScript


In our new programming language, we've introduced a more intuitive and concise syntax for for loops. This simplified syntax reduces the need for manually specifying loop conditions and increments, making the code easier to write and understand.

Basic Syntax: 


for(blo i = 0 to 5) {
    // Loop body
}

  • blo i = 0: Declares the loop variable i using the blo keyword, which is block-scoped and mutable.
  • to 5: Specifies the end value for the loop. The loop will iterate from 0 to 5 (inclusive) automatically incrementing i by 1 on each iteration.
  • Loop Body: The code inside the curly braces {} will execute for each value of i.

Example: Single Loop

Here’s an example of a single loop using the simplified syntax:

VisionScript
Try Yourself 

for(blo i = 1 to 5) {
    console.log("Iteration: " + i);
}

// Output:
// Iteration: 1
// Iteration: 2
// Iteration: 3
// Iteration: 4
// Iteration: 5

Example: Nested Loop

You can also use this simplified syntax for nested loops. Here’s an example:

VisionScript
Try Yourself 

for(blo i = 1 to 3) {
    console.log("Outer loop iteration: " + i);
    
    for(blo j = 1 to 2) {
        console.log("  Inner loop iteration: " + j);
    }
}

// Output:
// Outer loop iteration: 1
//   Inner loop iteration: 1
//   Inner loop iteration: 2
// Outer loop iteration: 2
//   Inner loop iteration: 1
//   Inner loop iteration: 2
// Outer loop iteration: 3
//   Inner loop iteration: 1
//   Inner loop iteration: 2
