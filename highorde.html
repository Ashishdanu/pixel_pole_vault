Array Methods: Filter, Sort, Find, Includes, and Reduces

Filter

Description: The Filter method creates a new array with all elements that pass the test implemented by the provided function.

Parameters:

  • array: The array on which to call the Filter method.
  • callback: A function that is executed on each element of the array.
  • thisArg (Optional): Value to use as this when executing the callback.

Return Value: A new array with elements that pass the test. If no elements pass, an empty array is returned.

VisionScript
Try This 

fix numbers = [1, 2, 3, 4, 5, 6];
// Use Filter to filter out even numbers
evenNumbers = numbers.Filter(function(number) {
    return number % 2 === 0;
});
console.log(evenNumbers); // Output: [2, 4, 6]

Sort

Description: The Sort method sorts the elements of an array in place and returns the sorted array.

Parameters:

  • array: The array on which to call the Sort method.
  • compareFunction (Optional): A function that defines the sort order.

Return Value: The sorted array. The sort is done in place, meaning that it modifies the original array.

VisionScript
Try This 

fix numbers = [5, 3, 8, 1, 2];
// Use Sort to sort numbers in ascending order
numbers.Sort();
console.log(numbers); // Output: [1, 2, 3, 5, 8]

// Use Sort with a custom compare function to sort in descending order
numbers.Sort(function(a, b) {
    return b - a;
});
console.log(numbers); // Output: [8, 5, 3, 2, 1]

Find

Description: The Find method returns the value of the first element in the array that satisfies the provided testing function.

Parameters:

  • array: The array on which to call the Find method.
  • callback: A function executed on each element until it finds one where the function returns true.

Return Value: The value of the first element that satisfies the testing function. If no elements satisfy the function, undefined is returned.

VisionScript
Try This 

fix numbers = [5, 12, 8, 130, 44];
// Use Find to find the first number greater than 10
found = numbers.Find(function(number) {
    return number > 10;
});
console.log(found); // Output: 12

Includes

Description: The Includes method determines whether an array includes a certain value among its entries, returning true or false.

Parameters:

  • array: The array on which to call the Includes method.
  • element: The element to search for within the array.
  • fromIndex (Optional): The position in the array to start the search. Defaults to 0.

Return Value: A boolean indicating whether the element is present in the array.

VisionScript
Try This 

fix numbers = [1, 2, 3, 4, 5];
// Check if array includes the number 3
includesThree = numbers.Includes(3);
console.log(includesThree); // Output: true

Reduces

Description: The Reduces method executes a reducer function on each element of the array, resulting in a single output value.

Parameters:

  • array: The array on which to call the Reduces method.
  • callback: A function executed on each element, taking four arguments: accumulator, currentValue, currentIndex, and sourceArray.
  • initialValue (Optional): A value to use as the first argument to the first call of the callback.

Return Value: The single value that results from the reduction.

VisionScript
Try This 

fix numbers = [1, 2, 3, 4, 5];
// Use Reduces to calculate the sum of all numbers
sum = numbers.Reduce(function(accumulator, currentValue) {
    return accumulator + currentValue;
}, 0);
console.log(sum); // Output: 15
