If-Else Conditions in the New Language


In your new programming language, conditional statements can be implemented using the standard if-else structure. This structure allows the code to execute certain blocks depending on whether a condition evaluates to true or false.

Basic Syntax: 


if (condition) {
    // Code to execute if condition is true
} else {
    // Code to execute if condition is false
}

Example 1: Simple Condition

Here is an example of a simple condition check:

VisionScript
Try Yourself 

blo x = 10;
if (x == 10) {
    console.log("x is 10");
} else {
    console.log("x is not 10");
}

// Output:
// x is 10

Example 2: Multiple Conditions

This example evaluates the age and categorizes it:

VisionScript
Try Yourself 

blo age = 20;
if (age < 13) {
    console.log("Child");
} else if (age >= 13 && age < 20) {
    console.log("Teenager");
} else {
    console.log("Adult");
}

// Output:
// Adult

Example 3: Nested If-Else

This example uses nested if-else statements:

VisionScript
Try Yourself 

blo num = 15;
if (num > 10) {
    if (num < 20) {
        console.log("num is between 10 and 20");
    } else {
        console.log("num is 20 or more");
    }
} else {
    console.log("num is 10 or less");
}

// Output:
// num is between 10 and 20

Key Features:

  • Human-Readable Syntax: The syntax is designed to be straightforward and intuitive, making it easy for developers to read and write code.
  • Standardized Structure: By adhering to the familiar if-else structure, the language ensures that users can transition smoothly from other programming languages.
