Array Methods: VSEvery and VSSome

VisionScript provides enhanced methods to evaluate array elements: VSEvery and VSSome. These methods are designed to simplify checks for array conditions, offering functionality similar to JavaScript's every and some methods.

1. VSEvery

Description: The VSEvery method tests whether all elements in the array pass the provided function. It returns true if all elements satisfy the callback condition; otherwise, it returns false.

VisionScript
Try Yourself 

    fix numbers = [2, 4, 6, 8, 10];
    // Use Every to check if all numbers are even
    allEven = numbers.VSEvery(function(number) {
        return number % 2 === 0;
    });
    console.log(allEven); // Output: true
    
    fix mixedNumbers = [1, 2, 3, 4, 5];
    // Use Every to check if all numbers are greater than 0
   allPositive = mixedNumbers.VSEvery(function(number) {
        return number > 0;
    });
    console.log(allPositive); // Output: true

2. VSSome

Description: The VSSome method tests whether at least one element in the array passes the provided function. It returns true if any element satisfies the callback condition; otherwise, it returns false.

VisionScript
Try Yourself 

    fix numbers = [1, 3, 5, 7, 9];
    // Use Some to check if there is any even number
    hasEvenNumber = numbers.VSSome(function(number) {
        return number % 2 === 0;
    });
    console.log(hasEvenNumber); // Output: false
    
    fix mixedNumbers = [1, 2, 3, 4, 5];
    // Use Some to check if there is any number greater than 4
    hasGreaterThanFour = mixedNumbers.VSSome(function(number) {
        return number > 4;
    });
    console.log(hasGreaterThanFour); // Output: true

String Methods: VSCharAt, VSLastIndexOf, VSToUpperCase, VSToLowerCase, VSConcat

1. VSCharAt

Description: This VSCharAt function retrieves the character at a specified index from a string, similar to JavaScript's charAt.

VisionScript
Try Yourself 

    fix str = "Hello, World!";
    // Use CharAt to get the character at a specific index
    fix charAt5 = str.VSCharAt(5);
    fix charAt11 = str.VSCharAt(11);
    console.log(charAt5); // Output: ","
    console.log(charAt11); // Output: "d"
    // Index out of range example
    fix charAt20 = str.VSCharAt(20);
    console.log(charAt20); // Output: ""

2. VSLastIndexOf

Description: The VSLastIndexOf method finds the last occurrence of a substring within a string.

VisionScript
Try Yourself 

    fix str = "Hello, World! Hello!";
    // Use LastIndexOf to find the last occurrence of a substring
    fix lastIndexOfHello = str.VSLastIndexOf("Hello");
    fix lastIndexOfWorld = str.VSLastIndexOf("World");
    console.log(lastIndexOfHello); // Output: 14 (the last "Hello" starts at index 14)
    console.log(lastIndexOfWorld); // Output: 7 (the "World" starts at index 7)
    // Search for a substring that doesn't exist
    fix lastIndexOfTest = str.VSLastIndexOf("Test");
    console.log(lastIndexOfTest); // Output: -1

3. VSToUpperCase

Description: The VSToUpperCase method converts all characters in a string to uppercase.

VisionScript
Try Yourself 

    fix str = "Hello, World!";
    // Use ToUpperCase to convert the string to uppercase
    fix upperStr = str.VSToUpperCase();
    console.log(upperStr); // Output: "HELLO, WORLD!"

4. VSToLowerCase

Description: The VSToLowerCase method converts all characters in a string to lowercase.

VisionScript
Try Yourself 

    fix str = "Hello, World!";
    // Use ToLowerCase to convert the string to lowercase
    fix lowerStr = str.VSToLowerCase();
    console.log(lowerStr); // Output: "hello, world!"

5. VSConcat

Description: The VSConcat method concatenates multiple strings.

VisionScript
Try Yourself 

    fix str1 = "Hello";
    fix str2 = " ";
    fix str3 = "World!";
    fix str4 = " How are you?";
    // Use Concat to concatenate multiple strings
    fix result = str1.VSConcat(str2, str3, str4);
    console.log(result); // Output: "Hello World! How are you?"

6. VSisAnagram()

Description: The VSisAnagram() method Checks if two strings are anagrams of each other. An anagram is a word or phrase formed by rearranging the letters of another.

VisionScript
Try Yourself 

    blo str1 = "listen";
    blo str2 = "silent";
    blo result = str1.VSisAnagram(str2); 
    console.log(result)
    // result is true

7. VScharacterFrequency()

Description: The VScharacterFrequency() returns a frequency map of characters in a string. The method counts how many times each character appears in the string and returns this information as an object (or map).

VisionScript
Try Yourself 

    blo str = "hello world";
    blo frequencyMap = str.characterFrequency();
    // frequencyMap = { h: 1, e: 1, l: 3, o: 2, ' ': 1, w: 1, r: 1, d: 1 }
